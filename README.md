# Aplocaciones-web-y-moviles- 2do Informatica 

repositorio oficial de la asignatura **Aplicaciones web y Moviles**


## Descripcion 
En esta asignatura los estudiantes desarrollaran habilidades para el diseño y construccion de aplicaciones web
El enfoque de la materia esta basada en: 

- Desarrollo practico de software
- -Construccion progresiva de aplicaciones reales
- trabajo colaborativo y resolucion de problema

## Objetivos de la asignatura
Al finalizar el curso, estudiante sera capaz de: 
- Crear interlaces funcionales
- Aplicar logica de programacion con JavaScript
- Desarrollar aplicacion interactivas
- Deiseñar interfaces responsivas
- Aplicar buenas practicas de desarrollo
